### 2. Какой запрос позволяет выбрать все записи из таблицы "students"?

WHERE
### 4. Какая команда объединяет строки из двух таблиц по общему полю?
JOIN

### 6. Какой запрос удалит таблицу с названием "employees"?
DROP TABLE employees;
